sap.ui.core.format.DateFormat,

formats and parses dates and times in JavaScript across 71 locales.

formats date with a given pattern with date field symbol(s).

parses date string with tolerance for manual user input.

Create an instance of DateFormat

Date Instance 


sap.ui.require([
    "sap/ui/core/format/DateFormat"
  ],
  function(DateFormat) {
    var oFormat = DateFormat.getDateInstance({
      // here define format options
    }, /*oLocale*/);
    var sDate = oFormat.format(new Date());
  }
);

Time Instance 



var oFormat = DateFormat.getTimeInstance({
  // here define format options
}, /*oLocale*/);

DateTime Instance 



var oFormat = DateFormat.getDateTimeInstance({
  // here define format options
}, /*oLocale*/);

sap.ui.core.format.NumberFormat,

formats and parses numbers in JavaScript across 71 locales.

formats number with default options for the current locale.

parses number string with tolerance for manual user input.

Create an instance of NumberFormat

Integer Instance 


sap.ui.require([
    "sap/ui/core/format/NumberFormat"
  ],
  function(NumberFormat) {
    var oFormat = NumberFormat.getIntegerInstance({
      // here define format options
    }, /*oLocale*/);
    var sNumber = oFormat.format(123456789);
  }
);

Float Instance 



var oFormat = NumberFormat.getFloatInstance({
  // here define format options
}, /*oLocale*/);

Percent Instance 



var oFormat = NumberFormat.getPercentInstance({
  // here define format options
}, /*oLocale*/);

Currency Instance 



var oFormat = NumberFormat.getCurrencyInstance({
  // here define format options
}, /*oLocale*/);

Define a pattern

#,##,##0.###

precision and decimals

precision The number of all digits (both integer and decimal).

123456 has precision 6

123456.789 has precision 9

decimals The number of the decimal digits.

123456 has 0 decimals

123456.789 has 3 decimals

precision precision: 3

1.2345 → 1.23

12.345 → 12.3

123.45 → 123

1234.5 → 1235

decimals decimals: 2

1.2345 → 1.23

123 → 123.00

Rounding

Native JavaScript Rounding 'toFixed()' 



1.005.toFixed(2) // "1.00" why not "1.01" ??
1.05.toFixed(2) // "1.1"
1.005 * 100 // 100.49999999999999

RoundingMode.TOWARDS_ZERO default for Integer instance

123.7 → 123

RoundingMode
 .HALF_AWAY_FROM_ZERO default for other instances

123.4 → 123

123.5 → 124

-123.5 → -124

Compact format

1,234,567,890 → 1.2B

set 'style: short'
or 'style: long'

short: 1.2M

long: 1.2 million

Scale factor is selected by default automatically

1234567 → 1.2M

1234567890 → 1.2B

Fixed scale factor use 'shortRefNumber'

shortRefNumber: 1000000

123456 → 0.12M

1234567 → 1.2M

1234567890 → 1,235M

Hide scale
'showScale: false' Get the scale 'getScale'

Currency

Number of decimals is chosen based on the currency 


var oFormat = NumberFormat.getCurrencyInstance({
}, /*oLocale*/);
oFormat.format(123.456, "EUR"); //"EUR 123.46"
oFormat.format(123.456, "JPY"); //"JPY 123"

Currency Code vs. Symbol 


var oFormat = NumberFormat.getCurrencyInstance({
  currencyCode: false
}, /*oLocale*/);
oFormat.format(-123.456, "EUR"); //"€﻿-123.46"

Currency with compact style 


var oFormat = NumberFormat.getCurrencyInstance({
  style: "short",
  currencyCode: false
}, /*oLocale*/);
oFormat.format(1234567, "EUR"); //"€1.2M"
